## 📌 Project Overview

The **Automated Car Parking System** is an Arduino-based robotics and IoT project designed to automate vehicle entry, exit, and parking slot monitoring. The system detects vehicles using IR sensors, verifies authorized users through RFID cards, controls a two-link robotic arm gate using a servo motor, displays available slots on an LCD, and sends parking records to an online server/database through the ESP-01S Wi-Fi module.



## ✨ Main Features

- Vehicle detection at entry and exit points using IR sensors
- RFID-based authorized access control
- Servo motor-controlled two-link robotic arm gate
- Real-time parking slot count update
- 16x2 I2C LCD display for system status
- ESP-01S Wi-Fi based IoT data transmission
- Online entry and exit record storage
- Low-cost academic prototype for robotics lab demonstration
